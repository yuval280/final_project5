Backup website container:
docker cp joomla-web:/var/www/html ./backup_html

Backup db:
docker exec joomla-mysql sh -c 'exec mysqldump --no-tablespaces -ujoomla -pjoomla joomla' > backup.sql


Restore for website container:
docker cp ./backup_html/. joomla-web:/var/www/html

Restore for db:
docker exec -i joomla-mysql mysql -ujoomla -pjoomla joomla < backup.sql
